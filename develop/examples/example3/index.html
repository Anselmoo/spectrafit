    Multi Peak Fitting

    In this example, the the input-file of SpectraFit is used to perform a multi peak fitting, shown in the figure below.

    Mulit peak fitting example by using 6 different pseudovoigt functions. The energy range is limited from -1 to 2.4 eV.

    All required settings are defined in the input-file in the section settings.

    settings

    If the data source, named infile is defined in the input-file, any filename can be used, like the undersocore _.

    Bash
    spectrafit _ -i example_2.json

    In this particular example, some of the pre-defined peaks are fixed. This is also printed in the terminal output:
