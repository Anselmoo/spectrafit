    Automatic Fitting

    In this example, the auto-peak-detection is introduced. The auto-peak-detection is a feature that automatically detects the peaks in the spectra. The routine is based on SciPy's find_peaks routine. As described in section advanced usage, the auto-peak-detection can be used via command line or explicit definitions of attributes in the section block of the input file.

    In the following example the in example the section block looks like:

    {
  "settings": {
    "column": ["Energy", "Intensity"],
    "decimal": ".",
    "energy_start": -1,
    "energy_stop": 4,
    "header": 0,
    "infile": "data.csv",
    "outfile": "example_7",
    "oversampling": false,
    "separator": ",",
    "shift": 0,
    "smooth": 0,
    "autopeak": {
      "model_type": "gaussian",
      "height": [0.0, 16],
      "threshold": [0.0, 16],
      "distance": 2,
      "prominence": [0.002, 12],
      "width": [0.002, 12],
      "wlen": 100
    },
    "verbose": 1,
    "version": false
  }
}

    This results in an automatic peak detection with the following results as shown in the figure shown below.

    Automatic detection of peaks in the spectra. In summary, 8 Gaussian distributions are used to fit this spectra.

    No found peaks

    In case no peaks are found, the routine will return an error and SpectraFit will not be able to continue.
