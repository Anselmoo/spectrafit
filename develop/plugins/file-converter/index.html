    File-Format-Conversion

    With the command spectrafit-file-converter input and also output files can be converted. The supported file formats are:

    1. .json
    2. .yaml or .yml
    3. .toml or .lock(output only)
    Bash
       ➜  spectrafit-file-converter -h
   usage: spectrafit-file-converter [-h] [-f {yml,lock,yaml,json,toml}] [-e {yml,lock,yaml,json,toml}] infile

   Converter for 'SpectraFit' input and output files.

   positional arguments:
     infile                Filename of the 'SpectraFit' input or output file.

   options:
     -h, --help            show this help message and exit
     -f {yml,lock,yaml,json,toml}, --file-format {yml,lock,yaml,json,toml}
                           File format for the conversion.
     -e {yml,lock,yaml,json,toml}, --export-format {yml,lock,yaml,json,toml}
                           File format for the export.

    Info

    The file format change can be also applied to the .json-output to the more human-readable lock-file-format, which is a .toml file.
